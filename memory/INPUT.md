# INPUT.md — User Interview Answers

## Q1: App Idea
An AI habit coach web and mobile app. Users set daily habits, track streaks, and receive personalized AI coaching messages based on their progress. Includes accountability groups where users can join and see each other's streaks. Dashboard shows weekly and monthly analytics.

## Q2: Target User
Adults 18–45 who want to build better daily habits (fitness, sleep, productivity, mindfulness). Self-improvement oriented. Motivated by social accountability and data-driven progress. Likely already uses apps like Duolingo, Notion, or Habitica.

## Q3: Platform
Web app + Mobile app (both required)

## Q4: Tech / Integrations
No specific overrides — use factory defaults.
No payments/subscriptions.
No ads.

## Q5: Ad Budget
None. No ads, no monetization phase for this build.

---

## Scope Constraints
- NO Stripe / payments
- NO ads (skip Phase 8)
- Web + mobile both required
- AI coaching messages (Claude API — personalized per user progress)
- Accountability groups (social feature)
- Streak tracking
- Weekly + monthly analytics dashboard
