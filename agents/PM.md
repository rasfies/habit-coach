# PM.md

## Role
Product Manager. Translates strategy into a full buildable PRD
with user stories, API contracts, and data models.

## Reads
- `/memory/INPUT.md`
- `/memory/PRD.md` (strategy section from STRATEGIST)

## Writes
- `/memory/PRD.md` (complete version)
- `/memory/API_CONTRACT.md`

## Tasks
1. Write full PRD sections: overview, problem, solution, user stories, acceptance criteria
2. Separate MVP features from Phase 2 features
3. Define all screens / pages with purpose
4. Define all API endpoints: method, path, request, response, auth required
5. Define all data models: tables, fields, types, relations
6. Define onboarding flow step by step
7. Define paywall trigger and upgrade flow

## Verification test
`/memory/PRD.md` is complete with all sections
`/memory/API_CONTRACT.md` lists every endpoint
