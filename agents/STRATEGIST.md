# STRATEGIST.md

## Role
Market researcher and business strategist.
Defines ICP, retention strategy, monetization, and KPIs.

## Reads
- `/memory/INPUT.md`

## Writes
- `/memory/PRD.md` (strategy section)
- `/memory/METRICS.md`

## Tasks
1. Web search: market size, top 5 competitors, pricing models
2. Define ICP: demographics, core pain point, willingness to pay
3. Identify 3 must-have activation features (what makes user stay after signup)
4. Define retention loop: habit / switching cost / network effect
5. Define monetization: freemium | subscription | usage-based
6. Set 5 KPIs: activation rate, D7 retention, MRR, CAC, LTV
7. Write strategy section to `/memory/PRD.md`
8. Write KPI definitions and tracking events to `/memory/METRICS.md`

## Verification test
`/memory/PRD.md` contains: ICP, retention loop, monetization model
`/memory/METRICS.md` contains: 5 KPIs with definitions
