## Tasks

### Landing page (`/content/landing-page.md`)
- Hero: headline (problem-focused), subheadline, CTA button text
- Problem section: 3 pain points with emotional language
- Solution section: 3 key features with benefit-first copy
- Social proof: placeholder for testimonials + metrics
- Pricing section: plan names, feature lists, CTA per plan
- FAQ: 5 questions addressing top objections
- Footer CTA

### Email sequence (`/content/emails/`)
- Day 0: Welcome + what to do first (activation nudge)
- Day 1: Core feature walkthrough
- Day 3: Social proof + use case story
- Day 7: Check-in + upgrade prompt if on free
- Day 14: Power user tip
- Day 21: Re-engagement if inactive
- Day 30: Success story + referral ask

### Ad copy (`/content/ad-copy.md`)
- 5 Meta ad variants: hook, body, CTA
- 5 Google ad variants: headline 1-3, description 1-2
- UGC video script brief (for Insense.pro production)
- Angle per variant: pain, social proof, transformation, curiosity, urgency

### SEO blog posts (`/content/blog/`)
- 3 posts targeting keywords identified by STRATEGIST
- 800-1200 words each, optimized for target keyword

## Verification test
All content files written and non-empty
Ad copy has 5 variants per platform
